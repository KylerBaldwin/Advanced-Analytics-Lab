# Advanced-Analytics-Lab
A comprehensive collection of analytics, ML, and AI projects completed as part of my Advanced Analytics coursework (Fall 2025). 

---

## Key Tools and Libraries
- **Python:** pandas, numpy, scikit-learn, xgboost, transformers, langchain, streamlit
- **Cloud:** Google Cloud (BigQuery, Vertex AI), Azure ML
- **Visualization:** matplotlib, seaborn, Data Studio  
- **Version Control:** Git + GitHub  

---

## Learning Objectives
- Apply matrix algebra and regression diagnostics in analytics.
- Build and deploy predictive models in GCP and Azure.
- Understand fundamentals of MLOps and ML model deployment.
- Develop and fine-tune LLMs using Hugging Face.
- Implement retrieval-augmented generation (RAG) pipelines.
